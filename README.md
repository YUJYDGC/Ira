# مختصر الروابط

تطبيق ويب بسيط لاختصار الروابط الطويلة وجعلها سهلة المشاركة.

## المميزات

- اختصار الروابط الطويلة بسهولة
- نسخ الرابط المختصر بنقرة واحدة
- عرض إحصائيات الرابط (عدد الزيارات، تاريخ الإنشاء)
- واجهة مستخدم بسيطة وجذابة باللغة العربية
- إعادة التوجيه التلقائي عند زيارة الرابط المختصر

## التقنيات المستخدمة

- الواجهة الأمامية: HTML, CSS, JavaScript
- الواجهة الخلفية: Node.js, Express
- قاعدة البيانات: MongoDB

## المتطلبات

- Node.js (الإصدار 14 أو أحدث)
- MongoDB (محلي أو عن طريق خدمة سحابية مثل MongoDB Atlas)

## التثبيت والتشغيل

1. **استنساخ المستودع**
   ```bash
   git clone <repository-url>
   cd url-shortener
   ```

2. **تثبيت الاعتماديات**
   ```bash
   npm install
   ```

3. **إعداد متغيرات البيئة**
   - إنشاء ملف `.env` في المجلد الرئيسي
   - نسخ محتويات ملف `.env.example` إلى ملف `.env`
   - تعديل قيم المتغيرات حسب الحاجة

4. **تشغيل التطبيق**
   ```bash
   # للتشغيل في وضع التطوير
   npm run dev

   # للتشغيل في وضع الإنتاج
   npm start
   ```

5. **فتح المتصفح**
   اذهب إلى `http://localhost:5000` في متصفحك
